 
## Tech Stack
- Socket.io
- Node.js with Express.js
- Qs
- EJS
- MDBootstrap
- Moment

## Install and run guide
Clone repository and install dependencies
```
git clone LINK
cd /folder
npm init
npm install
```
Start a server
```
node index.js
```
Go to your localhost and enjoy
```
http://localhost:3000
```
